# Xodor 评委测试指南

> 本指南帮助评委老师快速体验 Xodor 的核心功能

## 🎯 5分钟快速测试

### 第1步：启动程序（30秒）

1. 下载并解压 Release 包
2. 双击运行 `CodePracticeSystem.exe`
3. 程序会自动加载预置的 CCF 题库（45道题，第31-39次认证）

### 第2步：配置 AI 服务（1分钟）

**已提供测试 API Key，无需自行申请！**

1. 点击菜单栏 **设置 → AI 配置**
2. 选择 **云端模式**
3. 填入以下信息：
   ```
   API URL: https://api.deepseek.com
   API Key: sk-1a8bae2865f1443c99b924ffd14c4252
   模型: deepseek-chat
   ```
4. 点击 **测试连接**（应显示"连接成功"）
5. 点击 **保存**

### 第3步：体验核心功能（3分钟）

#### 功能1：AI 判题（1分钟）

1. 在左侧题库列表选择任意题目（如"坐标变换"）
2. 在代码编辑器中编写代码（或使用示例代码）
3. 点击 **AI 判题** 按钮
4. 观察 AI 实时分析代码并给出反馈

**测试要点**：
- ✅ AI 能否正确分析代码逻辑
- ✅ 是否提供优化建议
- ✅ 流式输出是否流畅

#### 功能2：AI 导师（1分钟）

1. 点击右侧 **AI 导师** 面板
2. 输入问题，如："这道题应该用什么算法？"
3. 观察 AI 导师的回答

**测试要点**：
- ✅ AI 能否理解题目上下文
- ✅ 回答是否专业且有帮助
- ✅ 支持代码片段引用

#### 功能3：智能题目导入（1分钟）

1. 点击菜单栏 **文件 → AI 智能导入**
2. 粘贴包含多个题目的文本（或选择测试文件）
3. 观察 AI 自动识别和拆分题目

**测试要点**：
- ✅ 能否正确识别题目边界
- ✅ 是否提取完整的题目信息
- ✅ 测试用例是否正确解析

## 🔧 常见问题

### Q1: AI 连接失败怎么办？

**A**: 检查以下几点：
1. 确认 API Key 正确：`sk-1a8bae2865f1443c99b924ffd14c4252`
2. 确认 API URL：`https://api.deepseek.com`
3. 检查网络连接
4. 点击"测试连接"按钮验证

### Q2: 题库没有显示怎么办？

**A**: 
1. 点击 **文件 → 题库管理**
2. 点击 **刷新** 按钮
3. 确认 `data/基础题库/CCF` 目录存在

### Q3: 代码编辑器没有语法高亮？

**A**: 
1. 确认文件扩展名正确（.cpp/.py/.java）
2. 在编辑器右下角选择语言
3. 重启程序

### Q4: AI 判题速度慢？

**A**: 
- 云端 API 响应时间取决于网络状况
- 通常 5-15 秒内完成
- 可以点击"终止输出"中断判题

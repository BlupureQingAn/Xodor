# 通讯延迟 (Communication Delay)

## 题目描述

给定平面上 n 个节点和 m 个基站。每个基站覆盖一个正方形区域（中心 (x_i, y_i)，边长 2r_i），区域内节点可以相互通讯，延迟为 t_i。求从节点 1 到节点 n 的最短通讯延迟。如果无法通讯，输出 'Nan'。
输入格式：第一行 n 和 m，接下来 n 行每行节点坐标，接下来 m 行每行基站坐标、半径和延迟。

## 测试数据

### 测试用例 1 - 基本功能测试：样例输入输出 (原始数据)

**输入：**
```
5 5
0 0
2 4
4 0
5 3
5 5
1 2 2 5
3 5 2 6
2 0 2 1
4 2 2 3
5 4 1 2
```

**输出：**
```
6
```

### 测试用例 2 - 边界条件：两个节点不在同一基站覆盖范围内，无法通讯 (原始数据)

**输入：**
```
2 1
0 0
10 10
5 5 1 1
```

**输出：**
```
Nan
```

### 测试用例 3 - 基本功能测试：通过两个基站中转 (原始数据)

**输入：**
```

0 0
5 5
10 10

8 8 3 2
```

**输出：**
```

```

**输入：**

**输出：**

**输出：**

### 测试用例 6 - 特殊情况：多个基站重叠覆盖，选择最短路径 (AI补充)

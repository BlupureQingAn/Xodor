# 矩阵重塑（其二）

## 题目描述

给定初始 n×m 矩阵 M，支持 t 次操作：1 p q（重塑为 p×q）、2 0 0（转置）、3 i j（查询元素）。依次执行操作，对每个查询输出结果。

## 测试数据

### 测试用例 1 - 基本测试：查询、重塑、再查询 (原始数据)

**输入：**
```
3 2 3
1 2
3 4
5 6
3 0 1
1 2 3
3 1 2
```

**输出：**
```
2
6
```

### 测试用例 2 - 基本测试：查询、转置、查询、重塑、查询 (原始数据)

**输入：**
```
3 2 5
1 2
3 4
5 6
3 1 0
2 0 0
3 1 0
1 3 2
3 1 0
```

**输出：**
```
3
2
5
```

### 测试用例 3 - 边界条件：1x1矩阵，查询和转置操作 (原始数据)

**输入：**
```
1 1 2
100
3 0 0
2 0 0
3 0 0
```

**输出：**
```
100
100
```

### 测试用例 4 - 特殊情况：重塑后查询，转置后查询 (AI补充)

**输入：**
```
2 3 4
1 2 3
4 5 6
1 3 2
3 2 1
2 0 0
3 1 2
```

**输出：**
```
6
3
```

### 测试用例 5 - 特殊情况：包含负数的矩阵，重塑后形状变化大 (AI补充)

**输入：**
```
2 2 3
-1 -2
-3 -4
3 1 1
1 4 1
3 2 0
```

**输出：**
```
-4
-2
```

### 测试用例 6 - 特殊情况：全零矩阵，仅查询 (AI补充)

**输入：**
```
2 2 1
0 0
0 0
3 0 0
```

**输出：**
```
0
```

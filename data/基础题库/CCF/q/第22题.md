# 字符串变换 (String Transformation)

## 题目描述

给定一个初始字符串 s（用两个 # 括起来），一个字符替换函数 f 由 n 个字符对 (ch1, ch2) 表示（即 f(ch1)=ch2），以及 m 个查询，每个查询包含一个正整数 k。要求对初始字符串 s 应用 k 次变换 F（将每个字符 ch 替换为 f(ch)），输出变换后的字符串（用两个 # 括起来）。未定义的 f(ch) 视为 f(ch)=ch。函数 f 是单射。
输入格式：第一行为初始字符串，第二行为 n，接下来 n 行每行为一个字符对，第 n+3 行为 m，第 n+4 行为 m 个查询的 k 值。

## 测试数据

### 测试用例 1 - 基本功能测试：样例输入输出 (原始数据)

**输入：**
```
#Hello World#
6
#HH#
#e #
# r#
#re#
#oa#
#ao#
3
1 2 3
```

**输出：**
```
#H llarWaeld#
#HrlloeWo ld#
#Hella Warld#
```

### 测试用例 2 - 基本功能测试：简单字符替换链 (原始数据)

**输入：**
```
#abc#
2
#ab#
#bc#
2
1 2
```

**输出：**
```
#bcc#
#ccc#
```

### 测试用例 3 - 边界条件：无字符对，变换任意次字符串不变 (原始数据)

**输入：**
```
#123#
0
1
1000000000
```

**输出：**
```
#123#
```

### 测试用例 4 - 特殊情况：单字符字符串，替换后稳定 (AI补充)

**输入：**
```
#a#
1
#ab#
3
1 2 3
```

**输出：**
```
#b#
#b#
#b#
```

### 测试用例 5 - 特殊情况：循环替换 (AI补充)

**输入：**
```
#abc#
3
#ab#
#bc#
#ca#
2
1 3
```

**输出：**
```
#bcc#
#aaa#
```

### 测试用例 6 - 特殊情况：空格字符替换为自身 (AI补充)

**输入：**
```
# #
1
#  #
1
5
```

**输出：**
```
# #
```

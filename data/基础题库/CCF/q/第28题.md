# 缓存模拟 (Cache Simulation)

## 题目描述

模拟一个 n-路组相联缓存的工作过程。缓存共有 N 个组，每组 n 个缓存行。内存块编号 k 只能存储在组 (k // n) % N 中。给定一系列读写指令（o=0 读，o=1 写，a 为内存块编号），使用 LRU 替换策略，输出实际对内存的读写操作序列（o'=0 读内存，o'=1 写回内存）。初始缓存为空。

## 测试数据

**输入：**
```
4 8 8

0 32
1 33
0 34
```

**输出：**
```

0 1
0 2
0 32
1 2
0 33
0 34
```

# 坐标变换（其一）

## 题目描述

对于平面直角坐标系上的坐标 (x, y)，定义了一个包含 n 个操作的序列 T = (t1, t2, ..., tn)。每个操作 ti 包含两个参数 dxi 和 dyi，表示将坐标 (x, y) 平移至 (x + dxi, y + dyi) 处。给定 m 个初始坐标，计算对每个坐标 (xj, yj) 依次进行 T 中 n 个操作后的最终坐标。

## 测试数据

### 测试用例 1 - 基本测试：样例输入 (原始数据)

**输入：**
```
3 2
10 10
0 0
10 -20
1 -1
0 0
```

**输出：**
```
21 -11
20 -10
```

### 测试用例 2 - 基本测试：单个操作 (原始数据)

**输入：**
```
1 1
5 -3
2 4
```

**输出：**
```
7 1
```

### 测试用例 3 - 边界条件：无操作 (原始数据)

**输入：**
```
0 2
1 2
3 4
```

**输出：**
```
1 2
3 4
```

### 测试用例 4 - 边界条件：最大绝对值输入，操作抵消 (AI补充)

**输入：**
```

```

**输出：**
```
0 0
```

### 测试用例 5 - 特殊情况：零操作 (AI补充)

**输出：**

### 测试用例 6 - 基本测试：连续正操作 (AI补充)

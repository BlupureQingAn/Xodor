# 化学方程式配平

## 题目描述

判断给定的化学方程式是否可以配平（即是否存在非零系数解）。化学式已化简为只包含小写字母和数字的字符串（不含括号）。
输入格式：第一行一个正整数 n。接下来 n 行，每行第一个整数 m 表示物质个数，随后 m 个字符串给出化学式。
输出格式：输出 n 行，每行 'Y' 或 'N' 表示是否可以配平。

## 测试数据

### 测试用例 1 - 基本测试：样例输入 (原始数据)

**输入：**
```
6
2 o2 o3
3 c1o1 c1o2 o2
2 n2o4 n1o2
4 cu1 h1n1o3 cu1n2o6 h2o1
4 al2s3o12 n1h5o1 al1o3h3 n2h8s1o4
4 c1o1 c1o2 o2 h2o1
```

**输出：**
```
Y
Y
Y
N
Y
Y
```

### 测试用例 2 - 基本测试：简单不可配平方程（H2 + O2） (原始数据)

**输入：**
```
1
2 h2 o2
```

**输出：**
```
N
```

### 测试用例 3 - 边界条件：单物质方程，总是可配平（系数任意） (原始数据)

**输入：**
```
1
1 h2o1
```

**输出：**
```
Y
```

### 测试用例 4 - 边界条件：三物质方程（H2 + O2 -> H2O），可配平 (AI补充)

**输入：**
```
1
2 h2 o2 h2o1
```

**输出：**
```
Y
```

### 测试用例 5 - 特殊情况：不同元素，无共同元素，不可配平 (AI补充)

**输入：**
```
1
2 ab1 bc1
```

**输出：**
```
N
```

### 测试用例 6 - 特殊情况：可配平（A + B -> AB） (AI补充)

**输入：**
```
1
3 a1 b1 a1b1
```

**输出：**
```
Y
```

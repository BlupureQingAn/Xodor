# 跳房子 (Hopscotch)

## 题目描述

有 n 个格子，第 i 个格子上的数字是 a_i（满足 a_i < i 且 a_n = 0）。从第 1 个格子开始。在第 i 个格子上最多能往前跳 k_i 格（不能超过第 n 格），即可以跳到 [i+1, min(n, i+k_i)] 中的任意格子 j。跳到格子 j 后，必须后退 a_j 格，最终停在格子 j - a_j。求从第 1 个格子到达第 n 个格子的最少跳跃次数。如果无法到达，输出 -1。

## 测试数据

### 测试用例 1 - 基本测试：样例1 (原始数据)

**输入：**
```
10
0 1 1 1 1 3 1 0 3 0
2 4 5 4 1 4 1 3 5 3
```

**输出：**
```
4
```

### 测试用例 2 - 基本测试：样例2，无法到达 (原始数据)

**输入：**
```
5
0 1 2 3 0
3 4 4 10 15
```

**输出：**
```
-1
```

### 测试用例 3 - 边界条件：n=2，一步到达 (原始数据)

**输入：**
```
2
0 0
1 1
```

**输出：**
```
1
```

### 测试用例 4 - 基本测试：无后退，简单前进 (AI补充)

**输入：**
```
3
0 0 0
1 1 1
```

**输出：**
```
2
```

### 测试用例 5 - 特殊情况：由于后退导致无法前进到终点 (AI补充)

**输入：**
```
5
0 1 0 2 0
1 1 1 1 1
```

**输出：**
```
-1
```

### 测试用例 6 - 边界条件：k_i 足够大，可以从1直接跳到n (AI补充)

**输入：**
```
6
0 0 0 0 0 0
5 5 5 5 5 5
```

**输出：**
```
1
```

### 测试用例 7 - 边界条件：最大规模n=10^5，每个格子只能跳一步，无后退，需要n-1步 (AI补充)

**输入：**
```
100000
0 0 0 ... 0
1 1 1 ... 1
```

**输出：**
```
99999
```

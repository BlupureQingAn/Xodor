# 梯度求解

## 题目描述

给定一个由常数、自变量和加法、减法、乘法组成的函数 f 的逆波兰式（后缀表达式），求函数在给定点处对指定自变量的偏导数，结果对 10^9+7 取模。

## 测试数据

### 测试用例 1 - 基本测试：样例输入 (原始数据)

**输入：**
```
2 2
x1 x1 x1 * x2 + *
1 2 3
2 3 4
```

**输出：**
```
15
3
```

### 测试用例 2 - 基本测试：常数函数 (原始数据)

**输入：**
```
1 1
5
1 10
```

**输出：**
```
0
```

### 测试用例 3 - 基本测试：自变量函数 (原始数据)

**输入：**
```
1 1
x1
1 5
```

**输出：**
```
1
```

### 测试用例 4 - 边界条件：零点求导 (AI补充)

**输入：**
```
2 1
x1 x2 *
1 0 0
```

**输出：**
```
0
```

### 测试用例 5 - 边界条件：大数输入，注意取模 (AI补充)

**输入：**
```
3 1
x1 x2 * x3 + 
1 100000 200000 300000
```

**输出：**
```
200000
```

### 测试用例 6 - 特殊情况：复合函数在零点的偏导 (AI补充)

**输入：**
```
2 1
x1 x1 * x2 x2 * +
1 0 0
```

**输出：**
```
0
```

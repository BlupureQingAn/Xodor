# 移动 (Movement)

## 题目描述

给定一个 n×n 的网格场地，机器人初始位置为 (x, y)。机器人可以接受由 'f'（前）、'b'（后）、'l'（左）、'r'（右）组成的指令序列移动。如果移动指令会使机器人移出场地边界，则机器人位置保持不变。处理 k 个查询，每个查询给出起始位置和指令序列，输出执行完指令后的最终位置。

## 测试数据

### 测试用例 1 - 基本测试：样例输入输出 (原始数据)

**输入：**
```
3 2
1 1 ffrrbbll
3 3 frbl
```

**输出：**
```
1 1
2 2
```

### 测试用例 2 - 基本测试：向前移动到达边界 (原始数据)

**输入：**
```
5 1
3 3 fff
```

**输出：**
```
3 5
```

### 测试用例 3 - 边界条件：连续向左移动超出边界，位置保持不变 (原始数据)

**输入：**
```
2 1
1 1 lll
```

**输出：**
```
1 1
```

### 测试用例 4 - 边界条件：场地大小为1，任何移动都无效 (AI补充)

**输入：**
```
1 1
1 1 f
```

**输出：**
```
1 1
```

### 测试用例 5 - 特殊情况：空指令序列，位置不变 (AI补充)

**输入：**
```
100 1
50 50 
```

**输出：**
```
50 50
```

### 测试用例 6 - 边界条件：向右移动到达边界 (AI补充)

**输入：**
```
10 2
1 5 rrrrrrrrrr
10 5 r
```

**输出：**
```
10 5
10 5
```

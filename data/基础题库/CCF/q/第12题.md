# 相似度计算

## 题目描述

给定两篇文章的单词列表，忽略大小写，计算两篇文章单词集合的交集大小和并集大小。
输入格式：第一行两个正整数 n 和 m。第二行 n 个单词（第一篇文章）。第三行 m 个单词（第二篇文章）。
输出格式：第一行输出 |A ∩ B|，第二行输出 |A ∪ B|。

## 测试数据

### 测试用例 1 - 基本测试：样例1，所有单词相同（忽略大小写） (原始数据)

**输入：**
```
3 2
The tHe thE
the THE
```

**输出：**
```
1
1
```

### 测试用例 2 - 基本测试：样例2，部分单词相同 (原始数据)

**输入：**
```
9 7
Par les soirs bleus dete jirai dans les sentiers
PICOTE PAR LES BLES FOULER LHERBE MENUE
```

**输出：**
```
2
13
```

### 测试用例 3 - 边界条件：单单词，大小写不同但视为相同 (原始数据)

**输入：**
```
1 1
apple
Apple
```

**输出：**
```
1
1
```

### 测试用例 4 - 边界条件：无交集 (AI补充)

**输入：**
```
2 2
cat dog
fish bird
```

**输出：**
```
0
4
```

### 测试用例 5 - 边界条件：空输入，两篇文章无单词 (AI补充)

**输入：**
```
0 0


```

**输出：**
```
0
0
```

### 测试用例 6 - 特殊情况：每篇文章有重复单词，但无交集 (AI补充)

**输入：**
```
3 3
a a a
b b b
```

**输出：**
```
0
2
```

# 数值积分

## 题目描述

给定二次函数 f(x) = x^2 + bx + c，其中 b 和 c 为整数。对于给定整数区间 [l, r]（l < r），选取区间内所有偶数坐标点（即 l ≤ x, x+2, x+4, ... ≤ r），计算每个点的函数值并求和，再乘以坐标点间距 2，得到估算的积分结果 s。
输入：一行四个整数 b, c, l, r。
输出：一个整数 s。
数据范围：0 ≤ l < l+2 ≤ r ≤ 1000，-20 ≤ b, c ≤ 20。

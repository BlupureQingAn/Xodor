# 木板切割 (Board Cutting)

## 题目描述

有一块长度为 n 的木板，分成 n 段，每段颜色为 c_i。进行 k 次切割操作，每次操作指定木板编号 x_i 和段落范围 [l_i, r_i]，切下该范围内的段作为新木板。每次操作后，输出切下木板中不同颜色数和颜色段数（极长连续相同颜色段）。
输入格式：第一行 n, m, k，第二行 n 个颜色值，接下来 k 行每行 x_i, l_i, r_i。

## 测试数据

### 测试用例 1 - 基本功能测试：样例输入输出 (原始数据)

**输入：**
```

1 2 2 3 1 2
1 3 4
1 5 5
1 4 5
1 1 6
2 4 4
```

**输出：**
```
2 2
1 1
0 0
2 2
1 1
```

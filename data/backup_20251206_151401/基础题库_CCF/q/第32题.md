# 机器人饲养指南

## 题目描述

有 n 个苹果，机器人一天最多吃 m 个苹果。每天投喂 i 个苹果获得快乐值 A_i（A_0=0）。投喂所有苹果，求最大总快乐值。
输入：第一行 n, m。第二行 m 个整数 A_1 到 A_m。
输出：一个整数，最大总快乐值。
数据范围：0 < n ≤ 10^4, 0 < m ≤ 100, 0 ≤ A_i ≤ 10^5。

## 测试数据

### 测试用例 1 - 基本测试：样例1，验证非单调收益下的最优分配。 (原始数据)

**输入：**
```
10 5
1 3 5 3 1
```

**输出：**
```
16
```

### 测试用例 2 - 基本测试：样例2，验证分配方案。 (原始数据)

**输入：**
```
4 3
1 60 100
```

**输出：**
```
120
```

### 测试用例 3 - 边界条件：n=0，没有苹果可喂，快乐值为0。 (原始数据)

**输入：**
```
0 5
1 2 3 4 5
```

**输出：**
```
0
```

### 测试用例 4 - 边界条件：m=1，每天只能喂1个苹果，每个快乐值100000，总快乐值为10000*100000=10^9。 (AI补充)

**输入：**
```
10000 1
100000
```

**输出：**
```
1000000000
```

### 测试用例 5 - 特殊情况：所有A_i=0，无论怎么喂快乐值都是0。 (AI补充)

**输入：**
```
5 3
0 0 0
```

**输出：**
```
0
```

### 测试用例 6 - 特殊情况：收益非单调，需要选择最优组合。例如：3+4=7个苹果，A_3=7, A_4=12，总19；或者2+2+3=7，A_2=10, A_2=10, A_3=7，总27？等等，计算：2+2+3=7，快乐值10+10+7=27，比19大。但需验证是否可行。 (AI补充)

**输入：**
```
7 4
5 10 7 12
```

**输出：**
```
21
```

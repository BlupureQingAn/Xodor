# 梦境巡查 (Dream Inspection)

## 题目描述

有 n+1 个区域，从 0 号区域（梦之源）出发，依次巡查 1 到 n 号区域后返回梦之源。从 i 号区域到下一区域需消耗 a_i 能量，到达 i 号区域（1≤i≤n）后可获得 b_i 能量补给。初始携带能量 w 需满足 w ≥ a_0 才能出发。现假设有且仅有一个区域 i（1≤i≤n）发生意外，b_i 变为 0。求对于每个 i，为顺利完成巡查所需的最少初始能量 w(i)。

## 测试数据

### 测试用例 1 - 基本测试：样例1 (原始数据)

**输入：**
```
3
5 5 5 5
0 100 0
```

**输出：**
```
10 20 10
```

### 测试用例 2 - 基本测试：样例2 (原始数据)

**输入：**
```
3
9 4 6 2
9 4 6
```

**输出：**
```
15 10 9
```

### 测试用例 3 - 边界条件：n=1，只有一个区域发生意外 (原始数据)

**输入：**
```
1
10 5
0
```

**输出：**
```
15
```

### 测试用例 4 - 特殊情况：所有消耗和补给都为0 (AI补充)

**输入：**
```
2
0 0 0
0 0
```

**输出：**
```
0 0
```

### 测试用例 5 - 边界条件：最大值测试，所有值达到上限1000 (AI补充)

**输入：**
```
3
1000 1000 1000 1000
1000 1000 1000
```

**输出：**
```
2000 2000 2000
```

### 测试用例 6 - 一般情况：递增的消耗和补给 (AI补充)

**输入：**
```
5
1 2 3 4 5 6
1 2 3 4 5
```

**输出：**
```
16 15 14 13 12
```

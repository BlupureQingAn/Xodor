# 机器人复健指南

## 题目描述

实验场地被划分为 n×n 个方格，机器人从 (x,y) 出发，可以向周围八个方向跳跃移动（国际象棋中马的走法：横向或纵向移动2格，另一方向移动1格）。若机器人只能跳动不超过 k 步，场地内有多少方格（包括起始位置）可以抵达？

## 测试数据

### 测试用例 1 - 基本测试：n=4, k=1, 起点(1,1)，可到达(1,1),(2,3),(3,2)。 (原始数据)

**输入：**
```
4 1
1 1
```

**输出：**
```
3
```

### 测试用例 2 - 基本测试：n=4, k=2, 起点(1,1)，可到达8个格子。 (原始数据)

**输入：**
```
4 2
1 1
```

**输出：**
```
8
```

### 测试用例 3 - 边界条件：n=1, 场地只有一个格子，只能待在起点。 (原始数据)

**输入：**
```
1 1
1 1
```

**输出：**
```
1
```

### 测试用例 4 - 边界条件：k=0，不能移动，只能待在起点。 (AI补充)

**输入：**
```
100 0
50 50
```

**输出：**
```
1
```

### 测试用例 5 - 特殊情况：n=3, k足够大，但场地小，所有格子均可到达。 (AI补充)

**输入：**
```
3 10
2 2
```

**输出：**
```
9
```

### 测试用例 6 - 特殊情况：k足够大，场地所有格子均可到达。 (AI补充)

**输入：**
```
100 100
1 1
```

**输出：**
```
10000
```

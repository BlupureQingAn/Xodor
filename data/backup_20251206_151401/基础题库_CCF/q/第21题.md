# 密码 (Password)

## 题目描述

给定 n 个密码字符串，每个密码由大写字母、小写字母、数字和特殊字符（* 和 #）组成，长度至少为 6。根据以下规则判断每个密码的安全级别：
- 高安全度（输出 2）：包含字母、数字和特殊字符，且同一个字符出现不超过 2 次。
- 中安全度（输出 1）：包含字母、数字和特殊字符，但未达到高安全度要求。
- 低安全度（输出 0）：未达到中安全度要求。
输入第一行为 n，接下来 n 行每行一个密码字符串。输出 n 行，每行一个整数表示安全级别。

# 正态分布

## 题目描述

对于正态分布随机变量 X ~ N(μ, σ²)，查表计算 P(X ≤ n)。首先将 X 转换为标准正态分布 Z = (X - μ) / σ，则 P(X ≤ n) = P(Z ≤ (n - μ) / σ)。题目假设有一张表，展示了 m 取值从 0.00 到 1.49 的结果（步长 0.01），其中每列对应 m 的百分位、每行对应 m 的十分位和整数部分。需要模拟查表过程，处理 k 个查询：对于给定的参数 μ、σ 和 n，计算 P(X ≤ n) 的结果在表中的哪一行、哪一列？行列下标均从 1 开始：行：0.0 对应第 1 行，0.1 对应第 2 行，依此类推……列：0.00, 0.01, …, 0.09 依次对应第 1、第 2 到第 10 列。

## 测试数据

### 测试用例 1 - 基本测试：μ=0, σ=1, n=0，Z=0.00，对应第1行第1列。 (原始数据)

**输入：**
```
1
0 1 0
```

**输出：**
```
1 1
```

```

```

**输入：**
```
1

```

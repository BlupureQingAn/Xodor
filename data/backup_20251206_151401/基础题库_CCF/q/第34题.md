# 集体锻炼

## 题目描述

n个同学站成一排，力量值a_i。定义区间[l,r]的体育价值为 l * r * gcd(a_l,...,a_r)。求所有区间的体育价值之和模998244353。
输入：第一行n，第二行n个整数a_i。
输出：一个整数，结果模998244353。
数据范围：1 ≤ n, a_i ≤ 10^6。

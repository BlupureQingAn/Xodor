# 文本分词

## 题目描述

给定 n 个单词及其频率，通过迭代合并相邻词汇对（按频率、长度、字典序优先级）生成词汇表，直到词汇表达到指定长度 m 或所有单词合并为一个词汇。按加入顺序输出词汇表。

## 测试数据

### 测试用例 1 - 基本测试：样例输入 (原始数据)

**输入：**
```
4 8
cut 15
cute 10
but 6
execute 3
```

**输出：**
```
b
c
e
t
u
x
ut
cut
```

### 测试用例 2 - 基本测试：单个单词，m大于字符种类数 (原始数据)

**输入：**
```
1 5
hello 1
```

**输出：**
```
e
h
l
o
hello
```

### 测试用例 3 - 边界条件：频率相同，测试优先级规则（长度、字典序） (原始数据)

**输入：**
```
3 6
aa 2
ab 2
ba 2
```

**输出：**
```
a
b
aa
ab
ba
aab
```

### 测试用例 4 - 特殊情况：m等于字符种类数，但过程会继续合并直到达到m (AI补充)

**输入：**
```
2 4
x 1
y 1
```

**输出：**
```
x
y
xy
xy
```

### 测试用例 5 - 特殊情况：高频单字符，合并过程可能产生长字符串 (AI补充)

**输入：**
```
3 10
a 100
b 50
c 25
```

**输出：**
```
a
b
c
ab
bc
abc
abcb
abcba
abcbab
abcbabc
```

### 测试用例 6 - 特殊情况：单词包含重复字符，初始词汇表可能少于m (AI补充)

**输入：**
```
2 3
zz 5
zzz 3
```

**输出：**
```
z
zz
zzz
```

# Xodor 数据目录说明

本目录包含 Xodor 智能刷题系统的数据文件。

## 📁 目录结构

```
data/
├── 基础题库/           # 预制题库（已包含在仓库中）
│   └── CCF/           # CCF 认证题库
│       ├── 31/        # 第31次认证题目（5题）
│       ├── 32/        # 第32次认证题目（5题）
│       ├── 33/        # 第33次认证题目（5题）
│       ├── 34/        # 第34次认证题目（5题）
│       ├── 35/        # 第35次认证题目（5题）
│       ├── 36/        # 第36次认证题目（5题）
│       ├── 37/        # 第37次认证题目（5题）
│       ├── 38/        # 第38次认证题目（5题）
│       └── 39/        # 第39次认证题目（5题）
│
├── sample_questions/  # 示例题目（已包含在仓库中）
│   ├── 01_两数之和.md
│   ├── 02_反转字符串.md
│   └── 03_斐波那契数列.md
│
├── user_answers/      # 用户代码（不包含在仓库中）
├── config/            # 配置文件（部分不包含在仓库中）
└── conversations/     # AI对话记录（不包含在仓库中）
```

## 📊 题库统计

### CCF 基础题库

- **总题数**: 45 道题
- **覆盖范围**: 第31-39次CCF认证（每次5题）
- **难度分布**: 
  - 简单题: 约 40%
  - 中等题: 约 45%
  - 困难题: 约 15%

### 题目格式

#### JSON 格式（原始题目）
```json
{
  "id": "unique-id",
  "title": "题目标题",
  "description": "题目描述",
  "input": "输入说明",
  "output": "输出说明",
  "samples": [
    {
      "input": "示例输入",
      "output": "示例输出"
    }
  ],
  "difficulty": "简单/中等/困难",
  "tags": ["标签1", "标签2"]
}
```

#### Markdown 格式（AI导入后）
```markdown
---
id: unique-id
title: 题目标题
difficulty: 简单/中等/困难
tags: [标签1, 标签2]
---

# 题目标题

## 题目描述
...

## 输入格式
...

## 输出格式
...

## 样例输入
...

## 样例输出
...
```

## 🚀 使用说明

### 1. 首次启动

程序首次启动时会自动扫描 `基础题库/` 目录，并注册所有题库。

### 2. 导入新题库

- **方式一**: 文件 → 导入题库 → 选择题目文件
- **方式二**: 文件 → AI 智能导入 → 自动识别和拆分题目

### 3. 题库管理

- 文件 → 题库管理
- 查看所有题库统计
- 刷新题库更新题目数量
- 删除不需要的题库

## 📝 注意事项

### 包含在Git仓库中的文件

- ✅ `基础题库/` - 预制题库
- ✅ `sample_questions/` - 示例题目
- ✅ `config.json` - 默认配置模板

### 不包含在Git仓库中的文件

- ❌ `user_answers/` - 用户的代码文件
- ❌ `conversations/` - AI对话历史
- ❌ `config/last_session.json` - 会话状态
- ❌ `config/user_progress.json` - 用户进度

这些文件是用户个人数据，不会被提交到Git仓库。

## 🔧 开发者说明

### 添加新题库

1. 在 `基础题库/` 下创建新目录
2. 放入题目文件（JSON或MD格式）
3. 程序会自动扫描并注册

### 题目文件命名规范

- JSON: `题目名称.json`
- Markdown: `题目名称.md` 或 `序号_题目名称.md`

### 配置文件

- `config.json` - 全局配置（AI设置、编辑器设置等）
- `config/CCF_parse_rule.json` - CCF题库导入规则

## 📚 题库来源

CCF 基础题库来源于中国计算机学会（CCF）认证考试真题，仅供学习和练习使用。

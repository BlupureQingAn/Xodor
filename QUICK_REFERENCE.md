# 智能题库导入系统 - 快速参考

## 🚀 快速开始

### 导入题库
```
1. 文件 → 导入题库 → AI智能导入
2. 选择题库文件夹
3. 输入题库名称
4. 等待导入完成
```

### 运行测试
```
1. 编写代码
2. 点击"▶ 运行测试"（F5）
3. 查看测试结果
4. 修改代码（如果需要）
5. 再次测试
```

## ✨ 核心功能

### 智能导入
- ✅ 自动拷贝文件到项目目录
- ✅ 智能拆分大文件（按题目边界）
- ✅ 逐块AI解析（支持大题库）
- ✅ 自动生成5+测试用例
- ✅ 详细进度显示
- ✅ 可随时取消

### 测试反馈
- ✅ LeetCode风格界面
- ✅ Accepted/Wrong Answer状态
- ✅ 每个用例详细信息
- ✅ 失败用例高亮显示
- ✅ 改进提示

## 📊 测试用例类型

每道题至少5个测试用例：
- **基本功能**（2-3个）：验证核心功能
- **边界条件**（1-2个）：空输入、最小值、最大值
- **特殊情况**（1-2个）：负数、零、重复、无解

## 💡 提示

### 题目文件格式
支持的分隔符：
- `# 题目标题`（一级标题）
- `1. 题目` / `1) 题目` / `1、题目`
- `第1题`
- `---` / `===` / `***`（分隔线）

### 最佳实践
1. 使用清晰的题目标题
2. 每道题包含完整描述
3. 提供示例输入输出
4. 标注难度和标签

### 常见问题

**Q: 导入失败怎么办？**
A: 检查AI服务是否运行（ollama serve），查看日志了解详情

**Q: 可以导入多大的题库？**
A: 无限制，系统会自动拆分大文件

**Q: 测试用例不够怎么办？**
A: AI会自动生成至少5个测试用例

**Q: 可以中途取消吗？**
A: 可以，点击"取消"按钮

## 📁 文件位置

- **题库目录**：`data/question_banks/<题库名>/`
- **用户代码**：`data/user_answers/<题目ID>.json`
- **错题本**：`data/wrong_questions.json`
- **进度记录**：`data/progress.json`

## 🔧 技术细节

### 文件拆分规则
- 小文件（<8KB）：不拆分
- 大文件（>=8KB）：按题目边界拆分
- 每块最大8KB
- 保证不切割题目

### AI提示词
- 要求生成5+测试用例
- 包含基本、边界、特殊情况
- 提供难度判断标准
- 智能识别题目信息

### 进度计算
```
进度 = (已处理块数 / 总块数) × 100%
```

# AI导师UI清理优化

## 修改内容

### 1. 移除AI导师面板内部的重复标题
**位置**：`src/ui/AIAssistantPanel.cpp`

**问题**：
- AI导师面板内部有一个"🤖 AI 导师"标题
- 但DockWidget已经有标题了，造成重复

**修改**：
- 移除内部的标题Label
- 保留"新对话"和"历史"按钮
- 将按钮改为水平排列，更紧凑

**修改前**：
```
┌─────────────────────────┐
│ 🤖 AI 导师  [🆕新对话][📜]│  ← 重复的标题
├─────────────────────────┤
│                         │
│   对话区域               │
│                         │
```

**修改后**：
```
┌─────────────────────────┐
│ [🆕 新对话] [📜 历史]    │  ← 直接显示功能按钮
├─────────────────────────┤
│                         │
│   对话区域               │
│                         │
```

### 2. 移除工具栏的错误列表按钮
**位置**：`src/ui/MainWindow.cpp`

**问题**：
- 工具栏有一个"🐛 错误列表"切换按钮
- 但错误列表已经设计为自动显示/隐藏
- 手动切换按钮是多余的

**修改**：
- 删除"🐛 错误列表"按钮及其相关代码
- 保留自动显示/隐藏逻辑：
  - 有错误时自动显示
  - 无错误时自动隐藏

**工具栏变化**：
```
修改前：[📚 题库管理] [🤖 AI导入题库] | [🐛 错误列表]
修改后：[📚 题库管理] [🤖 AI导入题库]
```

## 优势

1. **减少视觉冗余**
   - 移除重复的标题，界面更清爽
   - AI导师面板标题由DockWidget提供即可

2. **简化工具栏**
   - 移除不必要的手动控制
   - 错误列表智能显示，无需手动切换

3. **更好的空间利用**
   - AI导师面板顶部空间更紧凑
   - 为对话内容留出更多空间

4. **一致的交互逻辑**
   - 错误列表完全自动化，符合IDE习惯
   - 用户无需关心错误面板的显示状态

## 修改文件
- `src/ui/AIAssistantPanel.cpp` - 移除内部标题，优化按钮布局
- `src/ui/MainWindow.cpp` - 移除错误列表切换按钮

# 模拟题库生成系统 - 快速使用指南

## 🚀 快速开始

### 第一步：准备题库

确保已经导入了题库作为参考：
```
菜单 → 题目 → 导入题库
```

### 第二步：打开模拟题管理

```
菜单 → 题目 → 模拟题库管理 (Ctrl+Shift+M)
```

### 第三步：分析题库

1. 选择题库分类（CCF / LeetCode / 自定义）
2. 点击"分析题库"按钮
3. 查看分析结果：
   - 每套题目数量
   - 时间限制
   - 难度分布
   - 主要知识点

### 第四步：生成模拟题

1. 设置生成数量（建议2-3套）
2. 点击"开始生成"
3. 等待生成完成（约2-3分钟）
4. 查看生成日志

### 第五步：使用模拟题

生成的模拟题保存在：
```
人工模拟题库/
├── ccf/
│   ├── 模拟题1/
│   │   ├── 第1题.md
│   │   ├── 第2题.md
│   │   ├── 第3题.md
│   │   ├── 第4题.md
│   │   └── 答题说明.md
│   └── 模拟题2/
│       └── ...
```

## 📋 功能说明

### 1. 题库分析

**分析内容：**
- 题目总数统计
- 难度分布（简单/中等/困难）
- 知识点分布（前10个最常见）
- 出题规律提取

**分析结果示例：**
```
📊 分析结果：
• 每套题目数：4 道
• 时间限制：180 分钟
• 难度分布：简单 30%，中等 50%，困难 20%
• 主要知识点：数组、字符串、动态规划、贪心、图论
```

### 2. 模拟题生成

**生成配置：**
- 题库分类：选择题库类型
- 生成数量：1-10套（建议2-3套）
- 自动生成：测试数据、答题说明

**生成内容：**
- 完整题目描述
- 输入输出格式说明
- 数据范围和约束
- 5-8组测试数据
- 答题说明文档

### 3. 模拟题管理

**支持操作：**
- 查看：查看模拟题详情
- 删除：删除不需要的模拟题
- 导出：导出模拟题（开发中）

## 🔍 常见问题

### Q1: 生成失败怎么办？

**检查清单：**
1. AI服务是否运行？
   ```bash
   # 检查Ollama
   curl http://localhost:11434/api/tags
   ```

2. 题库是否为空？
   - 确保已导入题库
   - 至少10道题

3. 网络是否正常？
   - 检查网络连接
   - 查看防火墙设置

### Q2: 生成的题目质量不好？

**改进方法：**
1. 增加参考题库数量
2. 使用更强的AI模型
3. 调整出题规则
4. 人工审核和修改

### Q3: 生成时间太长？

**优化建议：**
1. 减少生成数量
2. 使用更快的AI模型
3. 检查系统资源
4. 分批生成

### Q4: 如何自定义出题规则？

**方法：**
1. 手动编辑规则文件：
   ```
   基础题库/ccf/出题模式规律.json
   ```

2. 修改参数：
   - questionsPerExam：每套题数
   - timeLimit：时间限制
   - difficultyRatio：难度比例
   - topicRatio：知识点比例

**原因：**

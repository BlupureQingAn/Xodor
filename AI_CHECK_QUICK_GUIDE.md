# AI连接检查 - 快速指南

## 🚀 新功能

程序启动时会自动检查AI服务连接状态！

## 📋 检查内容

✅ Ollama服务是否运行  
✅ 配置的模型是否已下载  
✅ 云端API是否可用（如果配置）  

## 💡 常见问题解决

### 问题1：提示"Ollama服务未运行"

**解决方法：**
```bash
# 在终端执行
ollama serve
```

### 问题2：提示"模型未安装"

**解决方法：**
```bash
# 下载模型（推荐qwen）
ollama pull qwen
```

### 问题3：想使用云端API

**解决方法：**
1. 打开 `工具 → 设置`
2. 切换到 `AI` 标签页
3. 输入API Key
4. 保存设置

## ⚠️ 重要提示

**AI服务不可用不影响核心功能！**

即使没有AI服务，您仍然可以：
- ✅ 刷题练习
- ✅ 编译运行代码
- ✅ 测试代码
- ✅ 查看历史记录
- ✅ 管理错题本

只有"AI代码分析"功能需要AI服务。

## 🎯 推荐配置

### 本地Ollama（推荐）
```bash
# 1. 安装Ollama
# 访问 https://ollama.ai

# 2. 启动服务
ollama serve

# 3. 下载模型
ollama pull qwen
```

### 云端API（备用）
在设置中配置API Key即可。

## 📊 状态说明

| 状态栏提示 | 含义 |
|-----------|------|
| AI服务已就绪 | ✅ 可以使用AI分析 |
| AI服务不可用 | ⚠️ 需要配置AI服务 |
| 正在检查AI服务连接... | 🔄 检查中 |

## 🔧 手动重新检查

如果配置好AI服务后想重新检查：
1. 重启程序（会自动检查）
2. 或者直接使用AI分析功能（会实时检测）

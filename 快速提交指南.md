# 快速提交指南 - 5 分钟完成

## 🚀 一键提交

### 步骤 1：运行提交脚本

```bash
.\提交到GitHub.bat
```

按照提示完成推送。

### 步骤 2：创建 Release

1. 访问：https://github.com/BlupureQingAn/Xodor/releases
2. 点击 "Create a new release"
3. 填写信息：
   - Tag: `v1.0.0`
   - Title: `Xodor v1.0.0 - 首次正式发布`
   - Description: 复制 `创建Release.md` 中的内容
4. 点击 "Publish release"

### 步骤 3：发送邮件

**收件人**：`zgsm@sangfor.com.cn`

**主题**：`【码上AI】技术挑战赛-Xodor Team-[你的学校]`

**正文**：复制 `提交检查清单.md` 中的邮件模板，填写你的信息

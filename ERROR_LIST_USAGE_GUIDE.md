# 错误列表使用指南

## 📍 位置
错误列表位于**代码编辑器底部**，类似VSCode的问题面板布局。

## 🎯 功能特性

### 1. 智能显示/隐藏
- **有错误时**：自动显示错误列表
- **无错误时**：自动隐藏错误列表
- **手动控制**：
  - 点击工具栏"🐛 错误列表"按钮
  - 使用快捷键 `Ctrl+Shift+M`

### 2. 折叠/展开
- 点击错误列表顶部的 **▼** 按钮折叠列表（只显示标题栏）
- 点击 **▶** 按钮展开列表（显示所有错误）
- 折叠后仍可看到错误统计信息

### 3. 错误信息显示
- **错误统计**：显示错误和警告的数量
  - ✅ 无错误：绿色提示
  - ❌ 有错误：红色提示，显示错误数量
  - ⚠️ 有警告：黄色提示，显示警告数量
- **详细列表**：每个错误显示
  - 行号和列号
  - 错误类型（错误/警告）
  - 错误消息

### 4. 快速跳转
- **点击错误项**：自动跳转到对应代码位置
- **光标定位**：精确定位到错误的行和列
- **自动滚动**：确保错误行在可见区域

### 5. AI修复功能
- **修复选中**：修复当前选中的错误
- **修复全部**：一键修复所有错误
- **智能启用**：需要AI服务连接才能使用

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+M` | 显示/隐藏错误列表 |
| 点击错误项 | 跳转到错误位置 |
| `▼/▶` 按钮 | 折叠/展开列表 |

## 🔧 使用场景

### 场景1：编写代码时
1. 在代码编辑器中编写代码
2. 停止输入1秒后，自动进行语法检查
3. 如果有错误，错误列表自动在底部显示
4. 错误行显示红色波浪线

### 场景2：修复错误
1. 查看错误列表中的错误信息
2. 点击错误项跳转到对应位置
3. 修改代码修复错误
4. 错误自动消失，列表更新

### 场景3：使用AI修复
1. 确保AI服务已连接
2. 在错误列表中选择要修复的错误
3. 点击"🔧 修复选中"按钮
4. AI分析错误并提供修复建议

### 场景4：隐藏错误列表
1. 如果觉得错误列表占用空间
2. 点击工具栏"🐛 错误列表"按钮
3. 或按 `Ctrl+Shift+M` 隐藏
4. 需要时再次显示

## 💡 使用技巧

### 技巧1：快速定位错误
- 不要逐行查找错误
- 直接点击错误列表中的错误项
- 自动跳转到错误位置

### 技巧2：折叠列表节省空间
- 编写代码时折叠错误列表
- 只保留标题栏显示错误统计
- 需要查看详情时再展开

### 技巧3：利用颜色区分
- 红色文本 = 错误（必须修复）
- 黄色文本 = 警告（建议修复）
- 优先修复错误，再处理警告

### 技巧4：批量修复
- 使用"AI修复全部"功能
- 一次性修复所有错误
- 节省时间提高效率

## 🎯 最佳实践

## 🐛 常见问题

### Q1: 错误列表不显示？
**A:** 检查以下几点：
- 是否有语法错误？
- 是否手动隐藏了错误列表？
- 按 `Ctrl+Shift+M` 显示

### Q2: 语法检查不工作？
**A:** 确认：
- 编译器是否正确配置？
- 在设置中检查编译器路径
- 重启程序试试

### Q3: AI修复按钮不可用？
**A:** 需要：
- AI服务已连接
- 选中了错误项
- 检查AI连接状态

### Q4: 错误列表占用太多空间？
**A:** 可以：
- 点击▼按钮折叠列表
- 或完全隐藏错误列表
- 需要时再显示

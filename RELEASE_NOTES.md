- ✅ **错题本** - 智能收集错题，支持复习
- ✅ **模拟考试** - AI 生成模拟题库
- ✅ **代码版本** - 自动保存代码历史版本

- 🎨 **现代化 UI** - 深色主题，VSCode 风格
- 📊 **题库列表** - 可视化进度，多维筛选
- 🤖 **AI 导师面板** - 实时对话，智能辅导
- 🐛 **错误列表** - 实时语法检查，一键修复

## 🚀 安装步骤

### 1. 下载并解压
```
下载: CodePracticeSystem-v1.5.0-Windows-x64.zip
解压到任意目录，如: C:\CodePracticeSystem\
```

### 2. 运行程序
```
双击 CodePracticeSystem.exe
```

### 3. 配置编译器（可选）
```
1. 点击"工具 → 设置"
2. 输入编译器路径，如: C:\mingw64\bin\g++.exe
3. 点击"保存"
```

### 4. 配置 AI（可选）
```
本地 Ollama:
1. 安装 Ollama
2. 运行: ollama serve
3. 运行: ollama pull qwen
4. 程序会自动连接

云端 API:
1. 点击"工具 → 设置"
2. 勾选"使用云端 API"
3. 输入 API Key
4. 点击"保存"
```

### 5. 开始使用
```
1. 点击"文件 → AI智能导入题库"
2. 选择题库文件夹（可先用示例题库测试）
3. 等待 AI 解析完成
4. 开始刷题！
```

## ❓ 常见问题

### Q: 程序无法启动？
A: 
- 确保解压了完整的文件夹
- 检查是否有杀毒软件拦截
- 尝试以管理员身份运行

### Q: 提示缺少 DLL？
A: 
- 重新解压压缩包
- 确保所有文件都在同一目录
- 不要单独复制 exe 文件

### Q: 编译失败？
A:
- 检查是否安装了 C++ 编译器
- 在设置中配置正确的编译器路径
- 确保编译器可以正常运行（命令行测试: `g++ --version`）

### Q: AI 功能无响应？
A:
- 检查 Ollama 是否运行: `ollama serve`
- 检查模型是否下载: `ollama list`
- 或切换到云端 API 模式

### Q: 题库导入失败？
A:
- 确保题库文件为 Markdown 格式
- 参考 sample_questions 中的示例格式
- 检查 AI 服务是否正常

### Q: 为什么没有控制台窗口了？
A:
- 这是正常的，正式版本隐藏了控制台窗口
- 如需调试，可以查看日志文件
- 或使用开发版本（保留控制台）

# 应用程序图标 - 快速参考

## 图标设计
- **主题**：酒红色 + X 与 C 组合
- **主体**：X 字母叠加在 C 字母上
- **尺寸**：256x256（多尺寸 ICO）
- **特点**：无圆圈包裹，简洁现代

## 文件位置
```
resources/
├── icon.svg          # SVG 源文件
├── icon.png          # PNG 图标 (256x256)
├── icon.ico          # Windows ICO (多尺寸)
└── app_icon.rc       # Windows 资源文件
```

## 快速命令

### 重新生成图标
```bash
py create_icon_pil.py
```

### 验证图标
```bash
verify_icon.bat
```

### 重新编译（应用图标）
```bash
cmake --build build/Qt_6_9_2_MinGW_64_bit-Debug --target CodePracticeSystem
```

### 部署
```bash
deploy_debug_manual.bat
```

## 验证方法
1. 打开文件资源管理器
2. 查看 `build/Qt_6_9_2_MinGW_64_bit-Debug/CodePracticeSystem.exe`
3. 应该显示酒红色 X 字母图标

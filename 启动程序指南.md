# 代码刷题系统 - 启动程序指南

## 🚀 快速启动

### 方法1: 使用部署版本（推荐）

```bash
# 进入部署目录并启动
cd deploy
.\CodePracticeSystem.exe
```

**优势**: 
- 包含所有依赖库
- 独立运行，不需要开发环境
- 启动最快

### 方法2: 使用构建版本

```bash
# 直接运行构建目录中的程序
.\build\CodePracticeSystem.exe
```

**注意**: 需要系统PATH中有Qt库路径

### 方法3: 双击启动

直接双击以下任一文件：
- `deploy\CodePracticeSystem.exe` （推荐）
- `build\CodePracticeSystem.exe`

---

## 🔧 如果程序无法启动

### 检查1: 缺少依赖库

如果提示缺少DLL文件，使用部署版本：
```bash
cd deploy
.\CodePracticeSystem.exe
```

### 检查2: Qt环境问题

如果使用build版本出错，设置Qt环境：
```bash
set PATH=F:\Qt\qt\6.10.0\mingw_64\bin;%PATH%
.\build\CodePracticeSystem.exe
```

### 检查3: 重新编译

如果程序版本过旧，重新编译：
```bash
.\compile.bat
```

## 🐛 常见问题

### Q: 程序样式显示不正确（最常见问题）
**A**: Qt资源文件缺失，运行修复脚本：
```bash
.\fix_qt_resources.bat
```
或手动修复：
```bash
F:\Qt\6.9.2\mingw_64\bin\windeployqt.exe --release --no-translations deploy\CodePracticeSystem.exe
```

### Q: 程序启动后立即关闭
**A**: 使用命令行启动查看错误信息：
```bash
cd deploy
.\CodePracticeSystem.exe
```

### Q: 提示找不到Qt库
**A**: 使用部署版本，或设置Qt环境变量

### Q: 界面显示异常
**A**: 检查系统DPI设置，建议使用100%缩放

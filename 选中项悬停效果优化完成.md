## 样式规范

### 选中状态
```css
QListWidget::item:selected {
    background-color: #660000;  /* 红色主题 */
    color: white;
    outline: none;
}
```

### 选中+悬停状态
```css
QListWidget::item:selected:hover {
    background-color: #880000;  /* 更浅的红色 */
    outline: none;
}
```

### 普通悬停状态
```css
QListWidget::item:hover {
    background-color: #2a2a2a;  /* 或其他灰色 */
}
```

## 效果说明

1. **统一红色主题**：所有选中项使用 `#660000` 红色
2. **悬停变浅**：鼠标悬停在选中项时，颜色变为 `#880000`（更浅的红色）
3. **移除圆框**：所有控件都添加 `outline: none`，移除选中时的圆框
4. **视觉一致性**：整个应用的列表控件样式完全统一
